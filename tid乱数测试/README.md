# tid乱数测试

这个目录是 `tid乱数测试` 的独立小项目。

## 入口文件

- `tid乱数测试.txt`
  只保留可调参数、名字设置和主流程调用。

## 模块说明

- `lib/flow.ecs`
  放流程动作和计时函数。
  包含函数：
  - `PreciseWait`：高精度等待指定毫秒数。
  - `启动与环境清理`：脚本启动后的初始按键流程。
  - `进入ID卡查询`：进入 ID 卡界面的固定动作流程。
  - `关闭软件`：关闭游戏软件并回到启动入口。

- `lib/id.ecs`
  放 TID 目标初始化、识图、匹配和结果处理。
  包含函数：
  - `初始化目标TID`：把 `_TARGET_TID` 拆成 `target1~target5`，并同步初始化 `seed`。
  - `识图`：读取 `ImgLabel` 中的五位数字标签，识别当前 TID。
  - `匹配`：判断是否命中目标 TID、豹子号、连号或 65535。
  - `处理ID卡结果`：串联识图、匹配、打印和循环控制状态赋值。

- `lib/name.ecs`
  放名字字符表和取名逻辑。
  包含函数：
  - `取名`：总入口，决定走默认 RED 还是自定义名字。
  - `取名RED`：输入默认名字 RED。
  - `自定义名称`：根据字符编码输入单个字符。
  - `向左移动光标`：名字输入界面横向移动。
  - `向上移动光标`：名字输入界面纵向移动。

- `lib/rng.ecs`
  放 RNG 推进逻辑。
  包含函数：
  - `NextSeed`：按 FRLG 的方式推进一次种子。

## 标签资源

- `lib/ImgLabel/`
  当前只保留这个项目实际引用到的标签：
  - `IDNo`
  - `防止SWITCH休眠`
  - `位置1图标0~9`
  - `位置2图标0~9`
  - `位置3图标0~9`
  - `位置4图标0~9`
  - `位置5图标0~9`

## 维护建议

- 改时序、目标和匹配规则：优先看 `tid乱数测试.txt`
- 改识图和匹配逻辑：看 `lib/id.ecs`
- 改流程动作：看 `lib/flow.ecs`
- 改名字输入：看 `lib/name.ecs`
